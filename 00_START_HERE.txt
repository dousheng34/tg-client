ШАГИ 1-3: ЛОКАЛЬНОЕ ТЕСТИРОВАНИЕ (5 минут)
─────────────────────────────────────────────

Шаг 1: Получить токен бота
  1. Откройте Telegram → найдите @BotFather
  2. Отправьте /newbot
  3. Следуйте инструкциям
  4. Скопируйте полученный токен

Шаг 2: Установить токен в bot.py
  1. Откройте файл bot.py
  2. Найдите строку: TOKEN = "YOUR_BOT_TOKEN_HERE"
  3. Замените на ваш токен: TOKEN = "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
  4. Сохраните файл

Шаг 3: Запустить бота локально
  1. Откройте терминал в папке проекта
  2. Выполните:
     source venv/bin/activate
     python bot.py
  3. Вы должны увидеть: "🚀 Бот іске қосылды!"
  4. Откройте Telegram и найдите вашего бота по username
  5. Отправьте /start и проверьте все функции

ШАГИ 4-5: РАЗВЕРТЫВАНИЕ НА СЕРВЕР (24/7)
──────────────────────────────────────────

Выберите один из двух вариантов:

ВАРИАНТ A: Render.com (РЕКОМЕНДУЕТСЯ - Бесплатно)
──────────────────────────────────────────────────
Простота: ⭐⭐⭐⭐⭐
Стоимость: Бесплатно
Время: 10 минут

Шаг 4a: Создать GitHub репозиторий
  1. Перейдите на https://github.com/new
  2. Создайте репозиторий "telegram-bot"
  3. Выполните в терминале:
     git init
     git add .
     git commit -m "Initial commit"
     git remote add origin https://github.com/YOUR_USERNAME/telegram-bot.git
     git push -u origin main

Шаг 5a: Развернуть на Render
  1. Перейдите на https://render.com
  2. Нажмите "New +" → "Web Service"
  3. Подключите GitHub репозиторий
  4. Заполните:
     - Name: telegram-bot
     - Runtime: Python 3
     - Build Command: pip install -r requirements.txt
     - Start Command: python bot.py
  5. Добавьте переменную окружения:
     - Key: BOT_TOKEN
     - Value: Ваш токен
  6. Нажмите "Deploy"

Готово! Бот работает 24/7 бесплатно.

ВАРИАНТ B: DigitalOcean (ЛУЧШИЙ ВЫБОР - $5/месяц)
──────────────────────────────────────────────────
Простота: ⭐⭐⭐⭐
Стоимость: $5/месяц
Надежность: ⭐⭐⭐⭐⭐
Время: 20 минут

Шаг 4b: Создать Droplet
  1. Перейдите на https://digitalocean.com
  2. Создайте Droplet:
     - Image: Ubuntu 22.04
     - Size: $5/месяц
  3. Получите IP адрес

Шаг 5b: Установить и запустить бота
  1. Подключитесь: ssh root@YOUR_SERVER_IP
  2. Установите зависимости:
     apt update && apt upgrade -y
     apt install -y python3 python3-pip python3-venv git
  3. Клонируйте репозиторий:
     cd /home
     git clone https://github.com/YOUR_USERNAME/telegram-bot.git
     cd telegram-bot
  4. Создайте виртуальное окружение:
     python3 -m venv venv
     source venv/bin/activate
     pip install -r requirements.txt
  5. Создайте systemd сервис (см. QUICK_START.md)
  6. Запустите:
     sudo systemctl start telegram-bot

Готово! Бот работает 24/7 на вашем сервере.

Q: Какой вариант развертывания выбрать?
A: Начните с Render.com (бесплатно), потом перейдите на DigitalOcean ($5/месяц)
   для большей надежности.

Q: Где взять токен бота?
A: Откройте Telegram → @BotFather → /newbot → скопируйте токен

Q: Бот не отвечает. Что делать?
A: 1. Проверьте токен в bot.py
   2. Проверьте интернет соединение
   3. Посмотрите логи: sudo journalctl -u telegram-bot -f

Q: Как обновить код бота?
A: git pull origin main && sudo systemctl restart telegram-bot

Q: Как добавить новые функции?
A: Отредактируйте bot.py и перезагрузите: sudo systemctl restart telegram-bot
