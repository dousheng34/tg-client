## 🚀 ОРНАТУ ПРОЦЕСІ

### Қысқа нұсқаулық (5 минут)

```bash
# 1. Репозиториялық клондау
git clone https://github.com/dousheng34/tg-client.git
cd tg-client

# 2. Модульдерді орнату
pip install -r requirements.txt

# 3. Токенді орнату
echo "BOT_TOKEN=YOUR_BOT_TOKEN_HERE" > .env

# 4. Ботты іске қосу
python kazakh_literature_bot.py
```

### Облақта развертывание (10 минут)

**Ұсынылатын платформа: Koyeb**

1. [Koyeb.com](https://www.koyeb.com) сайтына кіріңіз
2. GitHub репозиториялық байланыстырыңыз
3. `BOT_TOKEN` орта айнымалысын орнатыңыз
4. Deploy батынын басыңыз
