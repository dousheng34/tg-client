- **9+ қызықты деректер** — Авторлар мен шығармалар туралы фактілер
- **6 тарау** — Әдебиеттің түрлі аспектілері

### 🎮 Ойындар Жүйесі
1. **Викторина** — 8+ сұрақ, 4 нұсқа, +10 ұпай

## 🚀 ҚАЛАЙ БАСТАУ?

### ҚАДАМ 1: Орнату (5 минут)

```bash
# 1. Python 3.7+ орнату
python --version

# 2. Репозиториялық клондау
git clone https://github.com/YOUR_USERNAME/tg-kazakh-literature-bot.git
cd tg-kazakh-literature-bot

# 3. Модульдерді орнату
pip install -r requirements.txt

# 4. Токенді орнату
# kazakh_literature_bot.py файлын ашыңыз
# TOKEN = "YOUR_BOT_TOKEN_HERE" жолын табыңыз
# Өзгертіңіз: TOKEN = "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
```

### ҚАДАМ 2: Ботты іске қосу (1 минут)

```bash
python kazakh_literature_bot.py
```

**Күтілетін нәтиже:**
```
✅ Бот іске қосылды!
🎮 Викторина ойындарын ойнау үшін /ойын командасын жіберіңіз
📚 Авторлар туралы білу үшін /авторлар командасын жіберіңіз
```

### ҚАДАМ 3: Telegram-та тестілеу (5 минут)

```
1. Telegram-да ботыңызды іздеңіз
2. /start командасын жіберіңіз
3. Мәзір түймелерін басыңыз
4. Командаларды сынап көріңіз
```

## 🎮 ОЙЫНДАРДЫ ОЙНАУ

### Викторина

**Қалай ойнау:**
1. `/ойын` командасын жіберіңіз
2. Сұрақ мен 4 нұсқа көрсетіледі
3. Дұрыс жауапты таңдаңыз (A, B, C, D)
4. Нәтиже көрсетіледі (+10 ұпай немесе 0)

**Мысал:**
```
🎮 ВИКТОРИНА #1
═══════════════════════════════════════════════════════════════

❓ «Батыр Баян» поэмасы қай жылы жазылды?

Жауапты таңда:
A) 1920
B) 1923 ✅ ДҰРЫС!
C) 1930
D) 1915
```

### Авторлар туралы білу

**Қалай:**
1. `/авторлар` командасын жіберіңіз
2. Барлық авторлар туралы ақпарат көрсетіледі
3. Әрбір авторда:
   - Аты
   - Өмір сүрген жылдары
   - Туған жері
   - Бағыты
   - Шығармалары

### Шығармалар туралы білу

**Қалай:**
1. `/шығармалар` командасын жіберіңіз
2. Барлық шығармалар туралы ақпарат көрсетіледі
3. Әрбір шығармада:
   - Аты
   - Авторы
   - Жылы
   - Жанры
   - Мазмұны
   - Идеясы

### Іздеу функциясы

**Қалай:**
1. Авторлар, шығармалар немесе кейіпкерлер атын жаз
2. Бот сізге ақпарат берем

**Мысалдар:**
```
Пайдаланушы: "Мағжан"
Бот: Мағжан Жұмабаев туралы толық ақпарат

Пайдаланушы: "Батыр Баян"
Бот: Батыр Баян шығармасы туралы толық ақпарат

Пайдаланушы: "Көксерек"
Бот: Көксерек кейіпкері туралы толық ақпарат
```
