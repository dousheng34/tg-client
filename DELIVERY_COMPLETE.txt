================================================================================

Для немедленного запуска:

1. Получить токен:
   Telegram → @BotFather → /newbot → скопировать токен

2. Установить токен:
   Открыть bot.py → найти TOKEN = "YOUR_BOT_TOKEN_HERE" → заменить

3. Запустить:
   source venv/bin/activate
   python bot.py

4. Протестировать:
   Открыть Telegram → найти вашего бота → протестировать

Готово! Бот работает локально.
