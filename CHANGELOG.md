## Версиялау

Бұл жоба [Semantic Versioning](https://semver.org/) пайдаланады:

- **MAJOR** версия: Үйлесімсіз өзгерістер
- **MINOR** версия: Артқа үйлесімді жаңа ерекшеліктер
- **PATCH** версия: Артқа үйлесімді қате түзетулер

Мысалы: `4.0.0`
- `4` = MAJOR версия
- `0` = MINOR версия
- `0` = PATCH версия

## Қалай ынамдастыруға болады?

Өзгерістерді ынамдастыруға қызығушылық танытсаңыз:

1. Репозиторийді форк етіңіз
2. Ветка жасаңыз (`git checkout -b feature/amazing-feature`)
3. Өзгерістер жасаңыз
4. Commit жасаңыз (`git commit -m 'feat: amazing feature'`)
5. Push жасаңыз (`git push origin feature/amazing-feature`)
6. Pull Request ашыңыз

Толық ынамдастыру құжаты үшін [CONTRIBUTING.md](CONTRIBUTING.md) қараңыз.
