## 🔄 Обновление бота

Когда вы хотите обновить код бота:

### Локально на вашем компьютере:

```bash
# 1. Отредактируйте bot.py
nano bot.py

# 2. Добавьте изменения в Git
git add .

# 3. Создайте коммит
git commit -m "Описание изменений"

# 4. Загрузите на GitHub
git push origin main
```

### На Koyeb:

- Koyeb **автоматически** обнаружит изменения на GitHub
- Начнется новое развертывание
- Бот обновится без вашего участия

Это называется **CI/CD** (Continuous Integration/Continuous Deployment).

## 🐛 Решение проблем

### Проблема: Приложение не запускается

**Признаки:**
- Статус "Failed" или "Crashed"
- Логи показывают ошибки

**Решение:**
1. Проверьте логи на Koyeb
2. Убедитесь, что `TELEGRAM_BOT_TOKEN` установлен правильно
3. Проверьте, что `bot.py` существует в репозитории
4. Убедитесь, что `requirements.txt` содержит все зависимости
5. Проверьте синтаксис Python кода

### Проблема: Бот не отвечает в Telegram

**Признаки:**
- Приложение работает (статус "Running")
- Но бот не отвечает на сообщения

**Решение:**
1. Проверьте статус приложения на Koyeb
2. Посмотрите логи для ошибок
3. Убедитесь, что токен правильный
4. Перезагрузите приложение (Redeploy)
5. Попробуйте отправить команду `/start`

### Проблема: Ошибка при развертывании

**Признаки:**
- Развертывание не завершается
- Логи показывают ошибки сборки

**Решение:**
1. Проверьте, что все файлы загружены на GitHub
2. Убедитесь, что `requirements.txt` правильный
3. Проверьте синтаксис Python кода
4. Посмотрите логи развертывания на Koyeb
5. Попробуйте перезагрузить приложение

### Проблема: Высокое использование памяти

**Признаки:**
- Приложение часто перезагружается
- Логи показывают ошибки памяти

**Решение:**
1. Увеличьте Memory на Koyeb до 512 MB
2. Проверьте код на утечки памяти
3. Перезагрузите приложение
