2️⃣ Получите BOT_TOKEN

   • Скопируйте токен

3️⃣ Создайте GitHub репозиторий

   • Нажмите Settings → Delete Service

6️⃣ Создайте новый сервис

   • Выберите GitHub и репозиторий
   • Заполните параметры

7️⃣ Добавьте переменные окружения

   • Нажмите "Create Service"
