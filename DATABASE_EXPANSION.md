## 🔧 Деректер Базасын Кеңейту

### 1. Жаңа Автор Қосу

`bot.py` файлында `AUTHORS` сөздігін ашыңыз және жаңа автор қосыңыз:

```python

    "name": "Авторды толық аты",
    "emoji": "📖",
    "years": "1900-1980",
    "bio": "Өмірбаян мәтіні. Авторды өмірі, білімі, ықпалы туралы.",
    "works": [
        "Шығарма 1",
        "Шығарма 2",
        "Шығарма 3"
    ],
    "influence": "Авторды әдебиетке еткен ықпалы",
    "quotes": [
        "Ұлы сөз 1",
        "Ұлы сөз 2"
    ]
}
```

**Мысалы:**

## 🔄 Деректер Базасын Синхрондау

### Қадам 1: Өзгерістерді Жасау

1. `bot.py` файлын ашыңыз
2. `AUTHORS`, `WORKS`, `CHARACTERS`, `FACTS` сөздіктеріне жаңа деректер қосыңыз
3. Файлды сақтаңыз

### Қадам 2: Өзгерістерді Тексеру

```bash
# Синтаксис қатесін тексеру
python -m py_compile bot.py

# Ботты сынау (жергілік)
python bot.py
```

### Қадам 3: Git-ке Жүктеу

```bash
# Өзгерістерді қосу
git add bot.py

# Коммит жасау
git commit -m "Add: New authors, works, characters from NotebookLM"

# GitHub-ке жүктеу
git push origin main
```
