ВАРИАНТ 3: РАЗВЕРНУТЬ НА RENDER.COM (БЫСТРО)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Загрузите на GitHub (см. Вариант 2)

2. Перейдите на https://render.com

3. Нажмите "New +" → "Web Service"

4. Подключите GitHub репозиторий

5. Заполните:
   - Name: telegram-bot
   - Runtime: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: python bot.py

6. Добавьте переменную окружения:
   - Key: BOT_TOKEN
   - Value: ваш токен

7. Нажмите Deploy

8. Бот работает 24/7 бесплатно!

ВАРИАНТ 4: РАЗВЕРНУТЬ НА DIGITALOCEAN ($5/месяц)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Создайте Droplet на https://digitalocean.com
   - Image: Ubuntu 22.04
   - Size: $5/месяц

2. Подключитесь: ssh root@YOUR_SERVER_IP

3. Установите зависимости:
   apt update && apt upgrade -y
   apt install -y python3 python3-pip python3-venv git

4. Клонируйте репозиторий:
   cd /home
   git clone https://github.com/YOUR_USERNAME/telegram-bot.git
   cd telegram-bot

5. Создайте окружение:
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

6. Создайте systemd сервис:
   sudo nano /etc/systemd/system/telegram-bot.service

   Вставьте:
   [Unit]
   Description=Telegram Bot
   After=network.target

   [Service]
   Type=simple
   User=root
   WorkingDirectory=/home/telegram-bot
   ExecStart=/home/telegram-bot/venv/bin/python bot.py
   Restart=always
   RestartSec=10
   Environment="BOT_TOKEN=YOUR_TOKEN_HERE"

   [Install]
   WantedBy=multi-user.target

7. Запустите:
   sudo systemctl daemon-reload
   sudo systemctl enable telegram-bot
   sudo systemctl start telegram-bot

8. Проверьте:
   sudo systemctl status telegram-bot

9. Бот работает 24/7!
