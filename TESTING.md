## 💾 СЕССИЯ САҚТАУ ТЕСТІЛЕУ

### Ұпайларды сақтау
```
1. /ойын → Викторина #1
2. Дұрыс жауап → +10 ұпай
3. /ойын → Викторина #2
4. Дұрыс жауап → +10 ұпай (барлығы: 20)
5. /ұпайлар → Барлық ұпайлар: 20
```

### Статистика сақтау
```
1. 5 викторина ойнау
2. 3 дұрыс жауап
3. /ұпайлар → 30 ұпай, 3/5 дұрыс, 60%
```
