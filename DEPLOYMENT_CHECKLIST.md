# ✅ Чек-лист развертывания Kazakh Literature Bot v4.0

## 📋 Предварительная подготовка

### Шаг 1: Получить BOT_TOKEN
- [ ] Открыть Telegram
- [ ] Найти @BotFather
- [ ] Отправить `/newbot`
- [ ] Ввести имя бота (например: KazLit_Bot)
- [ ] Ввести username (например: kazlit_bot_2026)
- [ ] **Скопировать токен** (выглядит так: `123456789:ABCdefGHIjklMNOpqrSTUvwxYZ`)
- [ ] Сохранить токен в безопасном месте

### Шаг 2: Подготовить GitHub репозиторий
- [ ] Создать аккаунт на GitHub (если нет)
- [ ] Создать новый репозиторий: `kazakh-literature-bot`
- [ ] Клонировать репозиторий локально
- [ ] Скопировать файлы из `/home/code`:
  - [ ] `kazakh_literature_bot.py`
  - [ ] `requirements.txt`
  - [ ] `Dockerfile`
  - [ ] `.env.example`
  - [ ] `README.md`
  - [ ] `QUICK_FIX_KOYEB.txt`
- [ ] Сделать первый commit: `git add . && git commit -m "Initial commit: v4.0"`
- [ ] Сделать push: `git push origin main`

### Шаг 3: Проверить локально (опционально)
- [ ] Создать виртуальное окружение: `python3 -m venv venv`
- [ ] Активировать: `source venv/bin/activate`
- [ ] Установить зависимости: `pip install -r requirements.txt`
- [ ] Создать `.env` файл с `BOT_TOKEN=your_token`
- [ ] Запустить: `python kazakh_literature_bot.py`
- [ ] Проверить в Telegram, что бот работает
- [ ] Остановить бота (Ctrl+C)

## 🚀 Развертывание в Koyeb

### Шаг 4: Удалить старое развертывание
- [ ] Открыть https://app.koyeb.com
- [ ] Войти в аккаунт
- [ ] Найти старый сервис (kazakh-lit-bot или похожее)
- [ ] Нажать на сервис
- [ ] Нажать Settings (⚙️)
- [ ] Нажать "Delete Service"
- [ ] Подтвердить удаление
- [ ] Дождаться удаления (1-2 минуты)

### Шаг 5: Создать новое развертывание
- [ ] На главной странице Koyeb нажать "Create Service"
- [ ] Выбрать "GitHub"
- [ ] Авторизоваться с GitHub (если требуется)
- [ ] Выбрать репозиторий `kazakh-literature-bot`
- [ ] Заполнить форму:
  - [ ] Service name: `kazakh-lit-bot-v4`
  - [ ] Builder: `Docker`
  - [ ] Dockerfile path: `./Dockerfile`
  - [ ] Port: `8080` (по умолчанию)

### Шаг 6: Добавить переменные окружения
- [ ] В разделе "Environment variables" нажать "Add variable"
- [ ] Ввести:
  - [ ] Name: `BOT_TOKEN`
  - [ ] Value: `(ваш токен от BotFather)`
- [ ] Нажать "Create Service"
- [ ] Дождаться развертывания (2-3 минуты)

### Шаг 7: Проверить статус
- [ ] Статус должен быть "Running" ✅
- [ ] Если статус "Failed" ❌:
  - [ ] Нажать на сервис
  - [ ] Посмотреть Logs
  - [ ] Проверить BOT_TOKEN
  - [ ] Попробовать пересоздать сервис

## 🔧 Если что-то не работает

- [ ] Посмотреть логи в Koyeb (Logs tab)
- [ ] Попробовать отправить `/start` еще раз

### Проблема: Ошибка при развертывании
- [ ] Проверить, что все файлы в репозитории
- [ ] Проверить, что Dockerfile правильный

- [ ] Попробовать пересоздать сервис
