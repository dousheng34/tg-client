# 📚 Қазақ Әдебиеті Боты - Полный Обзор Проекта

## 📋 Обзор

Проект "Қазақ Әдебиеті Боты" - это полнофункциональный Telegram бот для изучения казахской литературы с полной локализацией на казахский язык.

**Версия:** 3.0  
**Статус:** ✅ Готово к продакшену  
**Дата завершения:** 8 апреля 2026  
**Платформа развертывания:** Koyeb (основная), Render (альтернатива)

## 🔄 Как Использовать Бот

### Вариант 1: Koyeb (Рекомендуется)

1. Перейти на [https://www.koyeb.com/](https://www.koyeb.com/)
2. Подключить GitHub репозиторий: `https://github.com/dousheng34/tg-client`
3. Установить переменную окружения: `TELEGRAM_BOT_TOKEN=ваш_токен`
4. Развернуть
5. Бот автоматически запустится

### Вариант 2: Render

1. Перейти на [https://render.com/](https://render.com/)
2. Подключить GitHub репозиторий: `https://github.com/dousheng34/tg-client`
3. Установить переменную окружения: `TELEGRAM_BOT_TOKEN=ваш_токен`
4. Развернуть
5. Бот автоматически запустится

### Вариант 3: Локально

```bash
git clone https://github.com/dousheng34/tg-client.git
cd tg-client
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
echo "TELEGRAM_BOT_TOKEN=ваш_токен" > .env
python bot.py
```
