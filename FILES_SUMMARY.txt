ЭТАП 1: ПОДГОТОВКА (5 минут)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Получить BOT_TOKEN:
   • Telegram → @BotFather → /newbot
   • Скопировать токен

2. Создать GitHub репозиторий:
   • Создать новый репозиторий: kazakh-literature-bot
   • Клонировать локально

3. Скопировать файлы:
   • Скопировать все файлы из /home/code в репозиторий
   • Сделать commit и push

ЭТАП 2: РАЗВЕРТЫВАНИЕ (5 минут)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4. Удалить старый сервис в Koyeb:
   • Открыть https://app.koyeb.com
   • Найти старый сервис
   • Нажать Settings → Delete Service

5. Создать новый сервис:
   • Нажать "Create Service"
   • Выбрать GitHub и репозиторий
   • Заполнить параметры

6. Добавить переменные окружения:
   • BOT_TOKEN = (ваш токен)
   • Нажать "Create Service"

ЭТАП 3: ПРОВЕРКА (5 минут)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

7. Дождаться развертывания:
   • Статус должен быть "Running" ✅
   • Это займет 2-3 минуты

8. Тестировать в Telegram:
   • Найти бота по имени
   • Отправить /start
   • Проверить все функции
   • Выбрать класс (1-11)
   • Попробовать игры
