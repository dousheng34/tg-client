## 🎯 Ботты қалай іске қосу?

### 1️⃣ ТОКЕН АЛАУ

1. Telegram-да **@BotFather** ботына жазыңыз
2. `/newbot` командасын жіберіңіз
3. Бот атын енгізіңіз (мысалы: `KazLitEncyclopedia`)
4. Username енгізіңіз (мысалы: `kazlit_edu_bot`)
5. **Токенді** көшіріп алыңыз

### 2️⃣ ТОКЕНДІ ОРНАТУ

`kazakh_literature_bot.py` файлын ашып, мына жолды өзгертіңіз:

```python
TOKEN = "YOUR_BOT_TOKEN_HERE"
```

Мысалы:
```python
TOKEN = "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
```

### 3️⃣ ОРНАТУ

```bash
# Виртуалды орта жасаңыз
python3 -m venv venv

# Активтеу
source venv/bin/activate  # Mac/Linux
# немесе
venv\Scripts\activate     # Windows

# Кітапханаларды орнатыңыз
pip install -r requirements.txt
```

### 4️⃣ ІСКЕ ҚОСУ

```bash
python kazakh_literature_bot.py
```

Сіз мына хабарламаны көруіңіз керек:
```
✅ Бот іске қосылды!
🎮 Викторина ойындарын ойнау үшін /ойын командасын жіберіңіз
📚 Авторлар туралы білу үшін /авторлар командасын жіберіңіз
```

## 🎮 ОЙЫНДАР

### 1. ВИКТОРИНА (Quiz)
- Сұрақ беріледі
- 4 нұсқа ұсынылады (A, B, C, D)
- Дұрыс жауап = **+10 ұпай** 🎉
- Қате жауап = **0 ұпай**

**Мысал сұрақтар:**
- «Батыр Баян» поэмасы қай жылы жазылды?
- Мұхтар Әуезовтің туған жері?
- «Көксерек» шығармасының жанры?

### 2. КІМ МЕН? (Кейіпкерді тап)
- Кейіпкер туралы белгілер беріледі
- Сіз кімдігін табуыңыз керек
- Дұрыс табу = **+10 ұпай**

### 3. АВТОР СӘЙКЕСТЕНДІРУ
- Шығармалар мен авторларды сәйкестендіру
- Дұрыс сәйкестендіру = **+10 ұпай**

### 4. ТОЛТЫРУ ОЙЫНЫ
- Сөйлемде бос орын қалдырылады
- Сіз сөзді толтырасыз
- Дұрыс толтыру = **+10 ұпай**

### 5. ҰПАЙ ЖҮЙЕСІ

**Рейтинг:**
- 0-30 ұпай: 📚 «Оқуды жалғастыр!»
- 40-70 ұпай: 👏 «Жақсы нәтиже!»
- 80-100 ұпай: 🏆 «Тамаша! Сен — шынайы білгір!»

#### 1. Мағжан Жұмабаев (1893–1938)
- **Туған жері:** Солтүстік Қазақстан, Сарыкөл ауданы
- **Бағыты:** Романтизм, символизм
- **Шығармалары:** «Батыр Баян», «Шолпанның күнәсі», «Ақ жол», «Қорқыт»
- **Өмірбаяны:** Қазақтың ұлы ақыны, символизм мен романтизм бағытында өзіндік мектеп қалыптастырды

#### 2. Мұхтар Әуезов (1897–1961)
- **Туған жері:** Шығыс Қазақстан, Абай ауданы
- **Бағыты:** Реализм, эпикалық проза
- **Шығармалары:** «Абай жолы», «Көксерек», «Еңлік-Кебек»
- **Өмірбаяны:** Қазақ прозасының негізін қалаушы, Лениндік сыйлықтың лауреаты

#### 3. Қасым Қайсенов (1918–2016)
- **Туған жері:** Семей облысы
- **Бағыты:** Реализм, автобиографиялық проза
- **Шығармалары:** «Жаутылындағы бала», «Өмір өткелдері»
- **Өмірбаяны:** Партизан-барлаушы, 98 жасқа дейін өмір сүрді

#### 4. Сансызбай Сарғасқаев (XX ғасыр)
- **Бағыты:** Балалар әдебиеті, юмор
- **Шығармалары:** «Тәмпіш қара»
- **Өмірбаяны:** Балалардың күнделікті өмірін юмормен суреттейтін жазушы

## 🔧 БОТТЫ ӨЗГЕРТУ

### Жаңа автор қосу

`kazakh_literature_bot.py` файлын ашып, `AUTHORS` сөздігіне қосыңыз:

```python
AUTHORS["new_author"] = {
    "name": "Авторды аты",
    "emoji": "📖",
    "years": "1900–1980",
    "born": "Туған жері",
    "bio": "Өмірбаян мәтіні...",
    "works_other": ["Шығарма 1", "Шығарма 2"],
    "style": "Жанр",
    "chapter": "balalar"
}
```

### Жаңа шығарма қосу

`WORKS` сөздігіне қосыңыз:

```python
WORKS["new_work"] = {
    "title": "Шығарма аты",
    "author": "author_id",
    "year": 1900,
    "genre": "Жанр",
    "summary": "Қысқа сипаттама...",
    "idea": "Идеясы...",
    "characters": ["char1", "char2"],
    "chapters": ["chapter1"]
}
```

### Жаңа викторина сұрағы қосу

`QUIZ_QUESTIONS` тізіміне қосыңыз:

```python
{
    "question": "Сұрақ мәтіні?",
    "options": ["Нұсқа 1", "Нұсқа 2", "Нұсқа 3", "Нұсқа 4"],
    "correct": 1,  # Дұрыс жауап индексі (0-3)
    "author": "author_id"
}
```
