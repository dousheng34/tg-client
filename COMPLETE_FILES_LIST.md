## 📄 ОПИСАНИЕ ОСНОВНЫХ ФАЙЛОВ

### 🤖 ОСНОВНОЙ КОД

#### `bot.py` (500+ строк)
- **Назначение:** Основной код Telegram бота
- **Функции:**
  - Обработка команд (/start, /help, /authors, /works, /characters, /chapters, /facts)
  - Обработка кнопок (20+ интерактивных кнопок)
  - Многопоточная архитектура (Main thread + HTTP health check)
  - Асинхронная обработка сообщений
  - Интеграция с базой данных
- **Язык:** Python 3.9+
- **Зависимости:** python-telegram-bot, python-dotenv, requests

#### `requirements.txt`
- **Назначение:** Список зависимостей Python
- **Содержимое:**
  ```
  python-telegram-bot==20.7
  python-dotenv==1.0.0
  requests==2.31.0
  ```

#### `.env`
- **Назначение:** Переменные окружения
- **Переменные:**
  - `TELEGRAM_BOT_TOKEN` - Токен Telegram бота

---

### 🚀 РАЗВЕРТЫВАНИЕ

#### `koyeb.yaml`
- **Назначение:** Конфигурация для развертывания на Koyeb
- **Функции:**
  - Автоматическое развертывание из GitHub
  - Установка переменных окружения
  - Конфигурация портов
  - Health check

#### `Procfile`
- **Назначение:** Конфигурация для развертывания на Render
- **Содержимое:**
  ```
  web: python bot.py
  ```

#### `runtime.txt`
- **Назначение:** Версия Python для Render
- **Содержимое:**
  ```
  python-3.11.8
  ```
