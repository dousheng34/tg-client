# 🚀 Руководство по развертыванию на Koyeb

## Что такое Koyeb?

**Koyeb** — это современная платформа для развертывания приложений с поддержкой:
- ✅ Бесплатного хостинга (до 2 приложений)
- ✅ Автоматического развертывания из GitHub
- ✅ 24/7 работы без перерывов
- ✅ Простого управления переменными окружения
- ✅ Встроенной поддержки Python

## 🎯 Пошаговое руководство

### Шаг 1: Создание аккаунта на Koyeb

1. Перейдите на [koyeb.com](https://www.koyeb.com)
2. Нажмите **"Sign Up"** (Зарегистрироваться)
3. Выберите **"Sign up with GitHub"** (Зарегистрироваться через GitHub)
4. Авторизуйте Koyeb доступ к вашему GitHub аккаунту
5. Завершите регистрацию

### Шаг 2: Создание нового приложения

1. На главной странице Koyeb нажмите **"Create App"** (Создать приложение)
2. Выберите **"GitHub"** как источник кода
3. Авторизуйте доступ к GitHub (если требуется)

### Шаг 3: Выбор репозитория

1. Найдите ваш репозиторий: **`dousheng34/tg-client`**
2. Нажмите на него для выбора
3. Выберите ветку: **`main`**

### Шаг 4: Конфигурация приложения

**Builder (Сборщик):**
- Выберите: **Python**
- Build command: оставьте пустым (по умолчанию)
- Run command: `python bot.py`

**Resources (Ресурсы):**
- Memory: **256 MB** (достаточно для бота)
- CPU: **100m** (минимум)

### Шаг 5: Переменные окружения

1. Нажмите **"Environment Variables"** (Переменные окружения)
2. Добавьте переменную:
   - **Key:** `TELEGRAM_BOT_TOKEN`
   - **Value:** `8167312154:AAFNFQqJ7_dwbbqFjlO4tTbpGSkD9lQP4rM`
3. Нажмите **"Add"** (Добавить)

### Шаг 6: Развертывание

1. Нажмите **"Create App"** (Создать приложение)
2. Koyeb начнет автоматическое развертывание
3. Дождитесь завершения (обычно 2-3 минуты)
4. Вы увидите статус **"Running"** (Работает)

## 🔄 Обновление бота

Когда вы обновляете код на GitHub:

1. Koyeb **автоматически** обнаружит изменения
2. Начнется новое развертывание
3. Бот обновится без вашего участия

Это называется **CI/CD** (Continuous Integration/Continuous Deployment).

## 🛠️ Управление приложением на Koyeb

### Просмотр логов

1. Откройте ваше приложение на Koyeb
2. Перейдите в **"Logs"** (Логи)
3. Смотрите вывод вашего бота в реальном времени

### Перезагрузка приложения

1. Откройте ваше приложение
2. Нажмите **"Redeploy"** (Переразвернуть)
3. Приложение перезагрузится

### Остановка приложения

1. Откройте ваше приложение
2. Нажмите **"Stop"** (Остановить)
3. Приложение перестанет работать

## 📊 Мониторинг

На Koyeb вы можете видеть:

- **Status** (Статус) — работает ли приложение
- **Logs** (Логи) — вывод вашего приложения
- **Metrics** (Метрики) — использование CPU и памяти
- **Deployments** (Развертывания) — история обновлений

## 🐛 Решение проблем

### Проблема: Приложение не запускается

**Решение:**
1. Проверьте логи на Koyeb
2. Убедитесь, что `TELEGRAM_BOT_TOKEN` установлен правильно
3. Проверьте, что `bot.py` существует в репозитории
4. Убедитесь, что `requirements.txt` содержит все зависимости

### Проблема: Бот не отвечает

**Решение:**
1. Проверьте статус приложения на Koyeb (должен быть "Running")
2. Посмотрите логи для ошибок
3. Убедитесь, что токен правильный
4. Перезагрузите приложение

### Проблема: Ошибка при развертывании

**Решение:**
1. Проверьте, что все файлы загружены на GitHub
2. Убедитесь, что `requirements.txt` правильный
3. Проверьте синтаксис Python кода
4. Посмотрите логи развертывания на Koyeb

## 💡 Советы и трюки

### Автоматическое развертывание

Koyeb автоматически развертывает приложение при каждом push на GitHub:

```bash
# Отредактируйте код локально
nano bot.py

# Загрузите на GitHub
git add .
git commit -m "Обновление бота"
git push origin main

# Koyeb автоматически развернет новую версию!
```

### Просмотр логов в реальном времени

На странице приложения в Koyeb нажмите **"Logs"** и смотрите вывод в реальном времени.

### Масштабирование

Если бот будет использоваться много пользователями:
1. Увеличьте Memory до 512 MB
2. Увеличьте CPU до 200m
3. Koyeb поддерживает горизонтальное масштабирование
