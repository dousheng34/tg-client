# 🚀 ОБНОВЛЕНИЕ БОТА В KOYEB (v4.0)

## ⚠️ ПРОБЛЕМА
В Koyeb развернута **старая версия бота (v1.0)** для 5-7 классов.
Нужно обновить на **v4.0** с полной базой 1-11 классов.

## ✅ РЕШЕНИЕ (5 минут)

### Шаг 1: Подготовить GitHub репозиторий

```bash
# Если еще нет репозитория:
cd /home/code
git init
git add .
git commit -m "Update to v4.0 - Full database 1-11 grades"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/kazakh-literature-bot.git
git push -u origin main
```

### Шаг 2: Удалить старое развертывание в Koyeb

1. Откройте https://app.koyeb.com
2. Найдите старое приложение (kazakh-lit-bot или похожее)
3. Нажмите **Settings** → **Delete Service**
4. Подтвердите удаление

### Шаг 3: Создать новое развертывание

1. На главной странице Koyeb нажмите **Create Service**
2. Выберите **GitHub** (если не подключен, подключите)
3. Выберите репозиторий `kazakh-literature-bot`
4. Заполните:
   - **Service name:** `kazakh-lit-bot-v4`
   - **Builder:** Docker
   - **Dockerfile path:** `./Dockerfile`
   - **Port:** 8000 (или оставить пусто для бота)

### Шаг 4: Добавить переменные окружения

В разделе **Environment variables** добавьте:

```
BOT_TOKEN=your_actual_bot_token_here
```

**Где получить BOT_TOKEN:**
1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте `/newbot`
4. Следуйте инструкциям
5. Скопируйте токен (выглядит как: `123456789:ABCdefGHIjklMNOpqrSTUvwxYZ`)

### Шаг 5: Развернуть

1. Нажмите **Create Service**
2. Ждите 2-3 минуты развертывания
3. Когда статус станет **Running** ✅, бот готов

### Шаг 6: Протестировать

1. Откройте Telegram
2. Найдите вашего бота по имени
3. Отправьте `/start`
4. Проверьте меню

## ❌ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Проблема: Бот не отвечает
- Проверьте BOT_TOKEN в переменных окружения
- Убедитесь, что токен скопирован полностью
- Перезагрузите сервис (Settings → Restart)

### Проблема: Ошибка при развертывании
- Проверьте логи (Logs tab)
- Убедитесь, что requirements.txt правильный
- Проверьте Dockerfile синтаксис

### Проблема: Старая версия все еще работает
- Удалите старый сервис полностью
- Очистите кэш браузера (Ctrl+Shift+Del)
- Создайте новый сервис с новым именем
