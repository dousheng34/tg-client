🔄 ОБНОВЛЕНИЕ БОТА ПОСЛЕ РАЗВЕРТЫВАНИЯ

Когда вы хотите обновить код:

1. Отредактируйте bot.py локально
2. Загрузите на GitHub:

   git commit -m "Описание изменений"
   git push origin main
3. Koyeb автоматически развернет новую версию!

Это называется CI/CD (Continuous Integration/Continuous Deployment).

🐛 РЕШЕНИЕ ПРОБЛЕМ

ПРОБЛЕМА: Бот не отвечает
РЕШЕНИЕ:
  1. Проверьте логи на Koyeb (App → Logs)
  2. Убедитесь, что TELEGRAM_BOT_TOKEN установлен правильно
  3. Перезагрузите приложение (Redeploy)

ПРОБЛЕМА: Ошибка при развертывании
РЕШЕНИЕ:
  1. Проверьте, что bot.py существует в репозитории
  2. Убедитесь, что requirements.txt содержит все зависимости
  3. Проверьте синтаксис Python кода

ПРОБЛЕМА: Переменные окружения не работают
РЕШЕНИЕ:
  1. Убедитесь, что переменная добавлена в Environment Variables
  2. Проверьте правильность имени переменной (TELEGRAM_BOT_TOKEN)
  3. Перезагрузите приложение после добавления переменной

Подробнее смотрите в KOYEB_FINAL_INSTRUCTIONS.md (раздел "Решение проблем")
