## 🚀 Развертывание

### Koyeb (Основная платформа)
```bash
1. Создать аккаунт на koyeb.com
2. Подключить GitHub репозиторий
3. Установить переменную окружения:
   TELEGRAM_BOT_TOKEN = ваш_токен
4. Развернуть
5. Бот автоматически запустится
```

### Render (Альтернатива)
```bash
1. Создать аккаунт на render.com
2. Подключить GitHub репозиторий
3. Установить переменную окружения:
   TELEGRAM_BOT_TOKEN = ваш_токен
4. Развернуть
5. Бот автоматически запустится
```

### Локальное развертывание
```bash
# Клонировать репозиторий
git clone https://github.com/dousheng34/tg-client.git
cd tg-client

# Создать виртуальную среду
python3 -m venv venv
source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt

# Создать .env файл
echo "TELEGRAM_BOT_TOKEN=ваш_токен" > .env

# Запустить бот
python bot.py
```
