## 📥 КАК НАЧАТЬ ИСПОЛЬЗОВАТЬ

### Шаг 1: Получить токен
```bash
# Перейти на https://t.me/BotFather
# Написать /newbot
# Следовать инструкциям
# Скопировать токен
```

### Шаг 2: Развернуть на Koyeb (Рекомендуется)
```bash
# 1. Перейти на https://www.koyeb.com/
# 2. Подключить GitHub репозиторий
# 3. Установить TELEGRAM_BOT_TOKEN
# 4. Нажать Deploy
```

### Шаг 3: Или развернуть на Render
```bash
# 1. Перейти на https://render.com/
# 2. Подключить GitHub репозиторий
# 3. Установить TELEGRAM_BOT_TOKEN
# 4. Нажать Deploy
```

### Шаг 4: Или запустить локально
```bash
git clone https://github.com/dousheng34/tg-client.git
cd tg-client
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
echo "TELEGRAM_BOT_TOKEN=ваш_токен" > .env
python bot.py
```
