## 🎮 ОЙЫНДАР ЖҮЙЕСІ

### 1️⃣ ВИКТОРИНА (Quiz)
**Сипаттамасы:** Сұрақ беріледі, 4 нұсқа ұсынылады

**Мысал сұрақтар:**
- «Батыр Баян» поэмасы қай жылы жазылды?
- Мұхтар Әуезовтің туған жері?
- «Көксерек» шығармасының жанры?
- Қасым Қайсенов неше жасқа дейін өмір сүрді?
- Мағжан Жұмабаев қай жылы ақталды?

**Ұпай жүйесі:**
- ✅ Дұрыс жауап = +10 ұпай
- ❌ Қате жауап = 0 ұпай

**Рейтинг:**
- 0-30 ұпай: 📚 «Оқуды жалғастыр!»
- 40-70 ұпай: 👏 «Жақсы нәтиже!»
- 80-100 ұпай: 🏆 «Тамаша! Сен — шынайы білгір!»

---

### 2️⃣ КІМ МЕН? (Кейіпкерді тап)
**Сипаттамасы:** Кейіпкер туралы белгілер беріледі, сіз кімдігін табасыз

**Мысал:**
```
Мен — жабайы табиғатта туып, адамдар арасында өстім. 
Екі дүние арасында қалдым. Мен кіммін?
→ Көксерек
```

**Ұпай:** Дұрыс табу = +10 ұпай

---

### 3️⃣ АВТОР СӘЙКЕСТЕНДІРУ
**Сипаттамасы:** Шығармалар мен авторларды сәйкестендіру

**Мысал:**
```
Батыр Баян → Мағжан Жұмабаев
Көксерек → Мұхтар Әуезов
Жаутылындағы бала → Қасым Қайсенов
```

**Ұпай:** Дұрыс сәйкестендіру = +10 ұпай

---

### 4️⃣ ТОЛТЫРУ ОЙЫНЫ
**Сипаттамасы:** Сөйлемде бос орын қалдырылады, сіз сөзді толтырасыз

**Мысал:**
```
Мағжан Жұмабаев ___ жылы дүниеге келді.
→ 1893

«Көксерек» шығармасы ___ жылы жазылды.
→ 1928
```

**Ұпай:** Дұрыс толтыру = +10 ұпай

---

### 5️⃣ ҰПАЙ ЖҮЙЕСІ
**Сипаттамасы:** Сессия бойынша ұпайларды сақтау және көрсету

**Функциялары:**
- ✅ Дұрыс жауап = +10 ұпай
- ❌ Қате жауап = 0 ұпай
- 📊 Барлық ұпайларды көру
- 📈 Дұрыс жауаптар процентін есептеу
- 🏆 Рейтинг беру
