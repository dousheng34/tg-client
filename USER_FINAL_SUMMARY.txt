🚀 КАК НАЧАТЬ ИСПОЛЬЗОВАТЬ БОТ?

ВАРИАНТ 1: РАЗВЕРТЫВАНИЕ НА KOYEB (РЕКОМЕНДУЕТСЯ)

Шаг 1: Получить токен от BotFather
  1. Перейти на https://t.me/BotFather
  2. Написать /newbot
  3. Следовать инструкциям
  4. Скопировать полученный токен

Шаг 2: Развернуть на Koyeb
  1. Перейти на https://www.koyeb.com/
  2. Подключить GitHub репозиторий: https://github.com/dousheng34/tg-client
  3. Установить переменную окружения: TELEGRAM_BOT_TOKEN=ваш_токен
  4. Нажать "Deploy"
  5. Бот автоматически запустится

ВАРИАНТ 2: РАЗВЕРТЫВАНИЕ НА RENDER

Шаг 1: Получить токен от BotFather (см. выше)

Шаг 2: Развернуть на Render
  1. Перейти на https://render.com/
  2. Подключить GitHub репозиторий: https://github.com/dousheng34/tg-client
  3. Установить переменную окружения: TELEGRAM_BOT_TOKEN=ваш_токен
  4. Нажать "Deploy"
  5. Бот автоматически запустится

ВАРИАНТ 3: ЛОКАЛЬНОЕ РАЗВЕРТЫВАНИЕ

  git clone https://github.com/dousheng34/tg-client.git
  cd tg-client
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  echo "TELEGRAM_BOT_TOKEN=ваш_токен" > .env
  python bot.py

Q: Как получить токен от BotFather?
A: Перейдите на https://t.me/BotFather, напишите /newbot и следуйте инструкциям.

Q: Какую платформу выбрать для развертывания?
A: Рекомендуется Koyeb (бесплатно, надежно). Render - хорошая альтернатива.

Q: Можно ли запустить бот локально?
A: Да, следуйте инструкциям в разделе "ВАРИАНТ 3: ЛОКАЛЬНОЕ РАЗВЕРТЫВАНИЕ".

Q: Как добавить новые авторы и произведения?
A: Отредактируйте файл bot.py, найдите раздел с базой данных и добавьте новые записи.

Q: Где найти полную документацию?
