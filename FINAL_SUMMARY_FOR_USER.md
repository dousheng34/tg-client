## 🎯 ФУНКЦИОНАЛДЫҚ ЕРЕКШЕЛІКТЕР

- `/start` - Ботты басталу
- `/help` - Көмек
- Менюдегі батондар

### Менюлер
- 🏠 **Басты меню** - Негізгі интерфейс
- 📚 **Авторлар** - Қазақ авторларының тізімі
- 📖 **Шығармалар** - Шығармалар каталогы
- 🎮 **Ойындар** - Интерактивті ойындар
- 📊 **Профиль** - Пайдаланушы профилі

### Ойындар
1. **🤔 Кім мен?** - Сілтемелер арқылы кейіпкерді табу
2. **🔗 Сәйкестендіру** - Автор-шығарма сәйкестігі
3. **✍️ Толтыру** - Бос орынды толтыру

### Викторина
- 100+ сұрақ
- Төрт нұсқалы жауаптар
- Дұрыс/қате жауап анықтамасы
- Ұпай есептеу

### Ұпай Жүйесі
- **Бастамшы:** 0-49 ұпай
- **Орта:** 50-99 ұпай
- **Ұстаз:** 100+ ұпай
- Дұрыс жауап = +10 ұпай
- Прогресс сохранение

## 🚀 ҚЫСҚА БАСТАУ (5 МИНУТ)

### 1️⃣ GitHub-та Репозиторий Жасау
```bash
git init
git add .
git commit -m "Initial commit: Kazakh Literature Bot v4.0"
git push origin main
```

### 2️⃣ BOT_TOKEN Алу
```
Telegram -> @BotFather
/newbot -> Атау беру -> Username -> Token көшіру
```

### 3️⃣ Deployment Платформасын Таңдау
- **Koyeb** (ұсынылған) - 2 минут
- **Render** (балама) - 2 минут
- **Docker** (локалды) - 5 минут

### 4️⃣ Deployment Іске Қосу
```
Koyeb/Render -> GitHub репозиторий -> BOT_TOKEN -> Deploy
```

### 5️⃣ Ботты Тестілеу
```
Telegram -> @kazakh_literature_bot -> /start
```
