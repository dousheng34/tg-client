## 🎯 Обзор проекта

**Қазақ Әдебиеті Боты (v3.0)** - это полнофункциональный Telegram бот, разработанный на Python с использованием Google NotebookLM для создания комплексной базы данных казахской литературы.

### Основные характеристики:

- ✅ **100% локализация на казахском языке**
- ✅ **Многопоточная архитектура** (Main polling + HTTP health check)
- ✅ **Полная база данных** (5 авторов, 5 произведений, 4 персонажа, 15+ фактов)
- ✅ **Развернуто на Koyeb** (основная платформа)
- ✅ **Альтернативное развертывание на Render**
- ✅ **Полная документация** (15+ файлов)
- ✅ **Готово к продакшену**

## 🚀 Как начать использовать

### Шаг 1: Получить токен от BotFather

1. Перейти на https://t.me/BotFather
2. Написать `/newbot`
3. Следовать инструкциям
4. Скопировать полученный токен

### Шаг 2: Развернуть на Koyeb (Рекомендуется)

1. Перейти на https://www.koyeb.com/
2. Подключить GitHub репозиторий: https://github.com/dousheng34/tg-client
3. Установить переменную окружения: `TELEGRAM_BOT_TOKEN=ваш_токен`
4. Нажать "Deploy"
5. Бот автоматически запустится

### Шаг 3: Или развернуть на Render

1. Перейти на https://render.com/
2. Подключить GitHub репозиторий: https://github.com/dousheng34/tg-client
3. Установить переменную окружения: `TELEGRAM_BOT_TOKEN=ваш_токен`
4. Нажать "Deploy"
5. Бот автоматически запустится

### Шаг 4: Или запустить локально

```bash
git clone https://github.com/dousheng34/tg-client.git
cd tg-client
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
echo "TELEGRAM_BOT_TOKEN=ваш_токен" > .env
python bot.py
```
