🚀 DEPLOYMENT ҮШІН 5 ҚАДАМ (15 минут)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ҚАДАМ 1: GitHub репозиторий жасаңыз (5 минут)
   
   a) GitHub-та жаңа репозиторий жасаңыз
   b) Төмендегі командаларды орындаңыз:
   
      git init
      git add .
      git commit -m "Initial commit: Kazakh Literature Bot v4.0"
      git remote add origin https://github.com/YOUR_USERNAME/kazakh-literature-bot.git
      git push -u origin main

ҚАДАМ 2: BOT_TOKEN алыңыз (2 минут)

   a) Telegram-да @BotFather-ге жазыңыз
   b) /newbot командасын жіберіңіз
   c) Ботқа атау беріңіз
   d) Username беріңіз
   e) Token көшіріңіз

ҚАДАМ 3: Deployment платформасын таңдаңыз

   ⭐ KOYEB (ҰСЫНЫЛҒАН) - 2 минут
      📄 KOYEB_DEPLOYMENT.md файлын оқыңыз
      
   📦 RENDER (БАЛАМА) - 2 минут
      📄 RENDER_DEPLOYMENT.md файлын оқыңыз
      
   🐳 DOCKER (ЛОКАЛДЫ) - 5 минут
      📄 DOCKER_GUIDE.md файлын оқыңыз

ҚАДАМ 4: Deployment іске қосыңыз (2 минут)

   Koyeb үшін:
      1. https://app.koyeb.com-ге кіріңіз
      2. GitHub репозиторийді байланыстырыңыз
      3. BOT_TOKEN құпиясын қосыңыз
      4. Deploy батырмасын басыңыз
      
   Render үшін:
      1. https://render.com-ге кіріңіз
      2. GitHub репозиторийді байланыстырыңыз
      3. BOT_TOKEN құпиясын қосыңыз
      4. Deploy батырмасын басыңыз

ҚАДАМ 5: Ботты тестіңіз (2 минут)

   a) Telegram-да ботты іздеңіз
   b) /start командасын жіберіңіз
   c) Меню пайда болуы керек
   d) Барлық команда жұмыс істейтінін тексеріңіз
