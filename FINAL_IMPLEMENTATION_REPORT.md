## 🚀 DEPLOYMENT

### Қолдау Ретінде Deployment Платформалары

#### 1. **Koyeb** (Ұсынылған)
```bash
# Қадамдар:
1. GitHub репозиторийін жасау
2. Koyeb.com-та сервис жасау
3. GitHub репозиторийін таңдау
4. BOT_TOKEN secret-ін орнату
5. Deploy
```

#### 2. **Render**
```bash
# Қадамдар:
1. GitHub репозиторийін жасау
2. Render.com-та сервис жасау
3. GitHub репозиторийін таңдау
4. BOT_TOKEN environment variable-ін орнату
5. Deploy
```

#### 3. **Docker**
```bash
# Локалды Docker-мен
docker build -t kazakh-literature-bot .
docker run -e BOT_TOKEN=your_token kazakh-literature-bot

# Docker Compose-мен
docker-compose up -d
```

#### 4. **GitHub Actions CI/CD**
```yaml
# Автоматты:
- Unit тестілер
- Код сканирлеу
- Docker image жасау
- Deployment
```
