## 🚀 ОРНАТУ ПРОЦЕСІ (5 минут)

### Қадам 1: Репозиториялық клондау
```bash
git clone https://github.com/dousheng34/tg-client.git
cd tg-client
```

### Қадам 2: Модульдерді орнату
```bash
pip install -r requirements.txt
```

### Қадам 3: Токенді орнату
```bash
echo "BOT_TOKEN=YOUR_BOT_TOKEN_HERE" > .env
```

### Қадам 4: Ботты іске қосу
```bash
python kazakh_literature_bot.py
```

- **Бос орындар:** 1-2
