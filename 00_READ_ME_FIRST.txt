   ⭐ Koyeb (ұсынылған) - 2 минут
   📦 Render (балама) - 2 минут
   🐳 Docker (локалды) - 5 минут

4️⃣ DEPLOYMENT ІСКЕ ҚОСУ:
   
   Koyeb/Render -> GitHub репозиторий -> BOT_TOKEN -> Deploy

5️⃣ БОТТЫ ТЕСТІЛЕУ:
   
   Telegram -> @kazakh_literature_bot -> /start
