   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   
   Откройте Telegram:
   • Найдите @BotFather
   • Отправьте: /newbot
   • Введите имя бота (например: KazLit_Bot)
   • Введите username (например: kazlit_bot_2026)
   • Скопируйте токен (выглядит так: 123456789:ABCdefGHIjklMNOpqrSTUvwxYZ)
   
   ⚠️ СОХРАНИТЕ ТОКЕН В БЕЗОПАСНОМ МЕСТЕ!

2️⃣ СОЗДАТЬ GITHUB РЕПОЗИТОРИЙ (3 минуты)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   
   На GitHub:
   • Создайте новый репозиторий: kazakh-literature-bot
   • Клонируйте его локально
   
   На вашем компьютере:
   • Скопируйте эти файлы из /home/code в репозиторий:
     - kazakh_literature_bot.py
     - requirements.txt
     - Dockerfile
     - .env.example
     - README.md
   
   • Сделайте commit:
     git add .
     git commit -m "Initial commit: v4.0"
     git push origin main

3️⃣ РАЗВЕРНУТЬ В KOYEB (5 минут)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   
   На Koyeb (https://app.koyeb.com):
   
   a) Удалить старый сервис:
      • Найдите старый сервис (kazakh-lit-bot или похожее)
      • Нажмите Settings → Delete Service
      • Подтвердите удаление
   
   b) Создать новый сервис:
      • Нажмите "Create Service"
      • Выберите "GitHub"
      • Выберите репозиторий kazakh-literature-bot
      • Заполните:
        - Service name: kazakh-lit-bot-v4
        - Builder: Docker
        - Dockerfile path: ./Dockerfile
   
   c) Добавить переменные окружения:
      • Нажмите "Add variable"
      • Name: BOT_TOKEN
      • Value: (ваш токен от BotFather)
      • Нажмите "Create Service"

4️⃣ ПРОВЕРИТЬ В TELEGRAM (5 минут)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   
   Когда статус в Koyeb станет "Running" ✅:
   
   • Откройте Telegram
   • Найдите вашего бота по имени
   • Отправьте /start
   • Проверьте меню (должны быть все опции)
   • Выберите класс (1-11)
   • Попробуйте выбрать автора
   • Попробуйте игру "Кім мен?"
   • Проверьте профиль

A: 2-3 минуты после нажатия "Create Service" в Koyeb

Q: Что если бот не работает?
A: Проверьте логи в Koyeb (Logs tab) и BOT_TOKEN

Q: Можно ли использовать старый токен?
A: Да, если у вас уже есть бот в @BotFather

A: Да, чтобы избежать конфликтов
