## 🚀 Быстрый старт

### Локальное развертывание

```bash
# 1. Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/kazakh-literature-bot.git
cd kazakh-literature-bot

# 2. Создать виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Создать .env файл
cp .env.example .env
# Отредактируйте .env и добавьте BOT_TOKEN

# 5. Запустить бота
python kazakh_literature_bot.py
```

### Развертывание в Koyeb (рекомендуется)

1. Создайте репозиторий на GitHub
2. Откройте https://app.koyeb.com
3. Нажмите "Create Service"
4. Выберите GitHub и ваш репозиторий
5. Добавьте переменную окружения: `BOT_TOKEN=your_token`
6. Нажмите "Create Service"

### Развертывание с Docker

```bash
# Собрать образ
docker build -t kazakh-lit-bot .

# Запустить контейнер
docker run -e BOT_TOKEN="your_token" kazakh-lit-bot
```

## 🔧 Конфигурация

### Переменные окружения

```env
# Обязательно
BOT_TOKEN=your_bot_token_from_botfather

# Альтернативно
TELEGRAM_BOT_TOKEN=your_bot_token_from_botfather
```

### Получить BOT_TOKEN

1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте `/newbot`
4. Следуйте инструкциям
5. Скопируйте токен
