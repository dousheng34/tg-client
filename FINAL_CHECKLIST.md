## 🎓 ИСПОЛЬЗОВАНИЕ

### Для пользователя
1. Получить токен от @BotFather
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить бот: `python bot.py`
4. Использовать команды на казахском языке

### Для разработчика
1. Клонировать репозиторий
2. Создать виртуальное окружение
3. Установить зависимости
4. Модифицировать базу данных в bot.py
5. Развернуть на Koyeb/Render
