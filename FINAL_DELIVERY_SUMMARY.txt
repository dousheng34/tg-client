Проект "Kazakh Literature Bot v3.0" - это полнофункциональный Telegram бот для
изучения казахской литературы, разработанный с использованием Google NotebookLM
для создания надежной базы данных авторов, произведений, персонажей и фактов.

УСТАНОВКА И ЗАПУСК:

1. Получить токен:
   • Откройте Telegram
   • Найдите @BotFather
   • Отправьте /newbot
   • Следуйте инструкциям
   • Скопируйте полученный токен

2. Подготовка окружения:
   • Клонируйте репозиторий: git clone https://github.com/dousheng34/tg-client.git
   • Перейдите в папку: cd tg-client
   • Создайте виртуальное окружение: python3 -m venv venv
   • Активируйте окружение: source venv/bin/activate (Mac/Linux)

3. Установка зависимостей:
   • pip install -r requirements.txt

4. Запуск бота:
   • Отредактируйте bot.py и вставьте токен
   • python bot.py

5. Использование:
   • Откройте бота в Telegram
   • Отправьте /start
   • Используйте команды на казахском языке
