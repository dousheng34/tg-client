      • Connect GitHub repository
      • Add BOT_TOKEN environment variable
      • Click Deploy
      
   🐳 DOCKER (Local - 5 minutes)
      • Run: docker build -t kazakh-lit-bot .
      • Run: docker run -e BOT_TOKEN="token" kazakh-lit-bot

   1. Find your bot on Telegram
   2. Send /start command
   3. Verify menu appears
   4. Test all commands
   5. Play a game
