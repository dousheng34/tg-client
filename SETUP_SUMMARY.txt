1️⃣  ЛОКАЛЬНОЕ ТЕСТИРОВАНИЕ (5 минут):
   
   a) Получить токен:
      - Откройте Telegram → @BotFather
      - Отправьте /newbot
      - Скопируйте токен
   
   b) Установить токен в bot.py:
      - Откройте bot.py
      - Замените: TOKEN = "YOUR_BOT_TOKEN_HERE"
      - На ваш токен: TOKEN = "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
   
   c) Запустить бота:
      source venv/bin/activate
      python bot.py
   
   d) Тестировать в Telegram:
      - Найдите вашего бота по username
      - Отправьте /start
      - Проверьте все функции

2️⃣  РАЗВЕРТЫВАНИЕ НА СЕРВЕР (выберите один вариант):

   ВАРИАНТ A: Render.com (РЕКОМЕНДУЕТСЯ - Бесплатно)
   ─────────────────────────────────────────────────
   Простота: ⭐⭐⭐⭐⭐
   Стоимость: Бесплатно
   
   Шаги:
   1. Создайте GitHub репозиторий
   2. Загрузите код (git push)
   3. Перейдите на render.com
   4. Нажмите "New +" → "Web Service"
   5. Подключите GitHub репозиторий
   6. Заполните:
      - Name: telegram-bot
      - Runtime: Python 3
      - Build Command: pip install -r requirements.txt
      - Start Command: python bot.py
   7. Добавьте переменную: BOT_TOKEN = ваш_токен
   8. Нажмите Deploy
   
   Готово! Бот работает 24/7 бесплатно.

   ─────────────────────────────────────────────────

   ВАРИАНТ B: DigitalOcean (ЛУЧШИЙ ВЫБОР - $5/месяц)
   ─────────────────────────────────────────────────
   Простота: ⭐⭐⭐⭐
   Стоимость: $5/месяц
   Надежность: ⭐⭐⭐⭐⭐
   
   Шаги:
   1. Создайте Droplet (Ubuntu 22.04, $5/месяц)
   2. Подключитесь: ssh root@YOUR_SERVER_IP
   3. Установите зависимости:
      apt update && apt upgrade -y
      apt install -y python3 python3-pip python3-venv git
   4. Клонируйте репозиторий:
      cd /home
      git clone https://github.com/YOUR_USERNAME/telegram-bot.git
      cd telegram-bot
   5. Создайте виртуальное окружение:
      python3 -m venv venv
      source venv/bin/activate
      pip install -r requirements.txt
   6. Создайте systemd сервис (см. QUICK_START.md)
   7. Запустите:
      sudo systemctl start telegram-bot
   
   Готово! Бот работает 24/7 на вашем сервере.
