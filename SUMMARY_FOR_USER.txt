🚀 КАК НАЧАТЬ ИСПОЛЬЗОВАТЬ БОТ:

ШАГИ:

1. Получить токен от BotFather
   • Перейти на https://t.me/BotFather
   • Написать /newbot
   • Следовать инструкциям
   • Скопировать полученный токен

2. Развернуть на Koyeb (Рекомендуется)
   • Перейти на https://www.koyeb.com/
   • Подключить GitHub репозиторий: https://github.com/dousheng34/tg-client
   • Установить переменную окружения: TELEGRAM_BOT_TOKEN=ваш_токен
   • Нажать "Deploy"
   • Бот автоматически запустится

3. Или развернуть на Render
   • Перейти на https://render.com/
   • Подключить GitHub репозиторий: https://github.com/dousheng34/tg-client
   • Установить переменную окружения: TELEGRAM_BOT_TOKEN=ваш_токен
   • Нажать "Deploy"
   • Бот автоматически запустится

4. Или запустить локально
   git clone https://github.com/dousheng34/tg-client.git
   cd tg-client
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   echo "TELEGRAM_BOT_TOKEN=ваш_токен" > .env
   python bot.py
