## 🔧 Техникалық Ақпарат

### Технологиялар
- **Python 3.8+**
- **python-telegram-bot 21.0** - Telegram бот API
- **aiohttp 3.9.1** - Асинхронды HTTP сервері
- **python-dotenv 1.0.0** - Орнату өзгерменінің басқарылуы

### Архитектура
- **Асинхронды бот** - Polling режимі
- **HTTP сервері** - Денсаулық тексеру үшін (8000 портында)
- **Мультиағынды** - Бот + HTTP сервері бөлек ағындарда

### Орнату Өзгерменері
```
TELEGRAM_BOT_TOKEN - Telegram бот токені (міндетті)
```
