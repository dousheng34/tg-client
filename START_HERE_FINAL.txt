Kazakh Literature Bot v3.0 - это полнофункциональный Telegram бот для изучения
казахской литературы. Бот содержит информацию о 5 авторах, 5 произведениях,
4 персонажах, 6 главах и 15+ интересных фактах.

✅ 100% локализирован на казахском языке
✅ Готов к развертыванию на Koyeb, Render или Heroku
✅ Полная документация включена
✅ Многопоточная архитектура для надежности

🚀 БЫСТРЫЙ СТАРТ (5 МИНУТ)
═══════════════════════════════════════════════════════════════════════════════

ШАГ 1: ПОЛУЧИТЬ ТОКЕН
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте команду: /newbot
4. Следуйте инструкциям:
   - Введите имя бота (например: "KazLitEncyclopedia")
   - Введите username (например: "kazlit_edu_bot")
5. Скопируйте полученный токен (выглядит как: 1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ)

ШАГ 2: ПОДГОТОВИТЬ ОКРУЖЕНИЕ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Клонируйте репозиторий:
   git clone https://github.com/dousheng34/tg-client.git
   cd tg-client

2. Создайте виртуальное окружение:
   python3 -m venv venv
   source venv/bin/activate  # Mac/Linux
   # или
   venv\Scripts\activate     # Windows

3. Установите зависимости:
   pip install -r requirements.txt

ШАГ 3: НАСТРОИТЬ БОТ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Откройте файл bot.py в текстовом редакторе
2. Найдите строку: TOKEN = "YOUR_BOT_TOKEN_HERE"
3. Замените на ваш токен: TOKEN = "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
4. Сохраните файл

ШАГ 4: ЗАПУСТИТЬ БОТ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Убедитесь, что виртуальное окружение активировано
2. Запустите бот:
   python bot.py
3. Вы должны увидеть сообщение: "Bot is running..."

ШАГ 5: ИСПОЛЬЗОВАТЬ БОТ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Откройте Telegram
2. Найдите вашего бота по username
3. Отправьте команду: /start
4. Используйте команды на казахском языке:
   • /авторлар - Авторы
   • /шығармалар - Произведения
   • /кейіпкерлер - Персонажи
   • /фактілер - Факты

    "born": "Туған жері",

    "works_other": ["Шығарма 1", "Шығарма 2"],

❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ
═══════════════════════════════════════════════════════════════════════════════

В: Как получить токен?
О: Найдите @BotFather в Telegram, отправьте /newbot и следуйте инструкциям.

В: Какие команды поддерживает бот?
О: /start, /авторлар, /шығармалар, /кейіпкерлер, /фактілер и другие.

В: Как развернуть бот на облачной платформе?
О: Прочитайте KOYEB_DEPLOYMENT_GUIDE.md или DEPLOYMENT_GUIDE.md.

В: Как добавить новых авторов?
О: Отредактируйте раздел AUTHORS в файле bot.py.

В: Какой язык используется?
О: 100% казахский язык. Все команды и сообщения на казахском.

В: Поддерживается ли Python 2?
О: Нет, требуется Python 3.7 или выше.

В: Как обновить бота?
О: Отредактируйте bot.py и перезапустите приложение.
