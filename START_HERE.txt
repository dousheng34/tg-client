🚀 БЫСТРЫЙ СТАРТ (3 ВАРИАНТА):

ВАРИАНТ 1: ЛОКАЛЬНОЕ ТЕСТИРОВАНИЕ (Самый быстрый)
─────────────────────────────────────────────────
1. Откройте терминал в папке проекта
2. Установите зависимости:
   pip install -r requirements.txt

3. Запустите бота:
   python bot.py

4. Откройте Telegram и найдите вашего бота
5. Нажмите /start

✅ Бот будет работать локально!


ВАРИАНТ 2: РАЗВЕРТЫВАНИЕ НА RENDER.COM (Рекомендуется - Бесплатно)
──────────────────────────────────────────────────────────────────
1. Загрузите проект на GitHub:
   - Следуйте инструкциям в GITHUB_PUSH_INSTRUCTIONS.md

2. Перейдите на https://render.com
3. Создайте аккаунт
4. Нажмите "New +" → "Web Service"
5. Подключите GitHub репозиторий
6. Заполните:
   - Name: kazakh-literature-bot
   - Runtime: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: python bot.py

7. Добавьте переменные окружения:
   - TOKEN = ваш Telegram токен
   - BOT_ID = ваш Bot ID

8. Нажмите "Create Web Service"

✅ Бот будет работать 24/7!
