## 🚀 Іске қосу

### 1️⃣ Токен алу

1. Telegram-да [@BotFather](https://t.me/BotFather) ботына жазыңыз
2. `/newbot` командасын жіберіңіз
3. Бот атын енгізіңіз (мысалы: `KazLitBot`)
4. Username енгізіңіз (мысалы: `kazlit_bot`)
5. **Токенді** көшіріп алыңыз

### 2️⃣ Орнату

```bash
# Репозиторийді клондау
git clone https://github.com/dousheng34/tg-client.git
cd tg-client

# Виртуалды орта жасау (ұсынылады)
python3 -m venv venv
source venv/bin/activate  # Mac/Linux
# немесе
venv\Scripts\activate     # Windows

# Кітапханаларды орнату
pip install -r requirements.txt
```

### 3️⃣ Орнату өзгерменін орнату

**.env файлын жасаңыз:**

```bash
TELEGRAM_BOT_TOKEN=your_token_here
```

**Немесе орнату өзгерменін орнатыңыз:**

```bash
export TELEGRAM_BOT_TOKEN="your_token_here"
```

### 4️⃣ Ботты іске қосу

```bash
python bot.py
```
